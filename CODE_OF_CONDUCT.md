# Code of Conduct

This project aims to be a welcoming, safe place for people to use, explore, and contribute to Hippocampus. This Code of Conduct describes the expectations for everyone who participates in the project and its community spaces (issue trackers, discussions, pull requests, and any related channels).

## Our Standards

We expect all participants to:

- Be respectful and considerate in all interactions.
- Assume good intent where possible and seek clarification before escalating.
- Welcome questions and different viewpoints.
- Give and accept constructive feedback.
- Use inclusive language and be mindful that others may have different backgrounds and experience levels.

The following behavior is not acceptable:

- Harassment, intimidation, threats, or personal attacks.
- Discrimination or derogatory comments related to gender, gender identity or expression, sexual orientation, disability, physical appearance, body size, race, ethnicity, age, religion, nationality, or any other protected characteristic.
- Sexualized language or imagery.
- Deliberate disruption of discussions or work.
- Publishing others’ private information (doxxing) without explicit permission.
- Any other conduct which could reasonably be considered inappropriate in a professional setting.

## Scope

This Code of Conduct applies within all project spaces, including but not limited to:

- Git hosting platforms (issues, pull requests, discussions, comments)
- Real-time chat or forum channels associated with the project
- Any in-person or virtual events officially organized for this project

It also applies when an individual is officially representing the project in public spaces.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, you are encouraged to report it. When reporting, please include:

- A description of what happened
- When and where it occurred (links, message excerpts, or screenshots if available)
- Any additional context you think may be helpful

At this time, reports should be sent directly to the project maintainers (for example via the contact information in the repository or project homepage). If you are unsure how to contact a maintainer, you may open a private issue or reach out via the platform’s direct messaging tools, if available.

We will treat all reports seriously and do our best to protect the privacy and safety of reporters.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may take any action they deem appropriate in response to behavior they consider unacceptable, including but not limited to:

- A private warning
- A request for a public or private apology
- Reverting, editing, or locking comments, issues, or pull requests
- Temporary or permanent bans from project spaces

Maintainers will strive to act fairly and consistently, but the primary goal is to protect the safety and well-being of participants and the health of the project.

## Amendments

This Code of Conduct may be updated from time to time to better reflect the needs of the project and its community. Substantive changes should be made transparently through the usual contribution process (for example, via a pull request) so they can be discussed and reviewed.

By participating in this project, you agree to abide by this Code of Conduct.
