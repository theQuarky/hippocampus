## Installation

### For Linux/macOS
You can simply run the following command to install:

```bash
sh install.sh
```

### For Windows
You can use the following command:

```powershell
.\\install.ps1
```

### Clone the Repository
To clone this repository, run:

```bash
git clone https://github.com/theQuarky/hippocampus
```
